# TravelerBackpack

**TravelerBackpack** — это плагин для Paper 1.21.1, вдохновлённый модом Traveler's Backpack. Он добавляет в игру удобный рюкзак с расширенными возможностями.

## Функции

- **Отдельный инвентарь** (36 слотов) — рюкзак можно носить в инвентаре.
- **Инструментальные слоты** — 4 слота для быстрого доступа к инструментам (кирка, топор, меч и т.д.).
- **Автосбор предметов** — с возможностью фильтрации (руды, еда, дерево).
- **Спальный мешок** — устанавливает время на рассвет, даёт эффекты, ограничение 1 раз в день.
- **Сортировка и фильтры** — в интерфейсе рюкзака.
- **Сохранение данных** — между перезапусками сервера.
- **Визуальные эффекты** — частицы при открытии и при "сне".

## Команды

- `/backpack` — открыть свой рюкзак.
- `/backpackgive <player>` — выдать рюкзак игроку (требует права `travelerbackpack.give`).

## Использование

1. Получите рюкзак командой `/backpackgive <ваш_ник>` (или через `/backpackgive <ник_игрока>`).
2. Щёлкните **ПКМ**, держа рюкзак в руке, чтобы открыть его.
3. Поместите инструменты в **инструментальные слоты** (в правом нижнем углу GUI).
4. При смене слота в хотбаре (1-9) — инструменты из рюкзака будут **временно выданы в руку**.
5. Используйте **спальный мешок**, чтобы "отдохнуть" ночью.
6. Используйте **фильтры** и **сортировку** для удобства.
